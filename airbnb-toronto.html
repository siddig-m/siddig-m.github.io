About

This project crafted an informative Airbnb dashboard for Toronto, highlighting key metrics. With 18,921 listings and 12,701 hosts, users can gauge the city's Airbnb landscape, featuring an average nightly price of $178. The dashboard's four main plots reveal popular neighborhoods, price distribution, host growth over time, and geographic listing distribution. Investors can identify competitive areas, and users can filter listings by type, with entire homes/apartments leading. This tool empowers hosts and investors with insights into Toronto's dynamic Airbnb market, aiding decision-making and property selection.
