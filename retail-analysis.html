About

This project revolves around analyzing retail data to understand factors influencing company-wide sales and predict future sales for the upcoming fiscal quarter. It explores relationships between sales and regional indicators like consumer price index and temperature, as well as internal data such as store size and product markdowns. Using an ARIMA time series model, the project aims to forecast department-specific sales, offering valuable insights for optimizing business performance. The analysis was conducted in a Jupyter Notebook using Python, leveraging data preprocessing and model selection techniques for accurate predictions.
