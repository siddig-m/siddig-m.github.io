About

With the prevalence of the COVID-19 and the impact it has had on our world today we wanted to contribute to the fight against the virus. We also wanted to create a project that will not only detect masks but also support the use of Personal Protective Equipment (PPE). Our project aims to classify if the user is wearing or not wearing a face-mask based on input from a photo or a live video using a Convolutional Neural Network (CNN).
